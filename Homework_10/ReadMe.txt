Для запуска  бота нужно создать файл "token.ini" в  директории 
\Homeworks\Homework_9\bin\Debug\netcoreapp3.1, добавить в файл 
токен в одну строку, без пробелов и лишних символов.

Для работы с ботом с клиентской стороны через телеграм:
1. Послать боту изображение.
2. Выбрать желаемый формат изобрадения, нажав  соответствующую 
кнопку из предлодженных.
/// Или отправить команду /start для получения  дополнительных
сведений.

Для работы с ботом со стороны серверной  части в главном  окне
приложения: можно просматривать историю  сообщений  в  текущей 
сессии и сохранять её в json. При этом результаты  предыдущего 
экспорта будут автоматически заархивированы.

Для отправки сообщения пользователю необходимо:
1. Получить сообщение от него.
2. Выделить сообщение.
3. Набрать текст в поле для текста.
4. Нажать кнопку отправить.

Для отправки файла:
1. Получить сообщение.
2. Выделить сообщение.
3. Нажать кнопку в виде скрепки.
4. Выбрать файл.
5. Нажать кнопку отправить.
